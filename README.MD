# Pov Digital Circus 
![plushie caine](plushieCaine.png)

## Requirements
 - g++
 - Raylib
 - Cmake

## Compiling and running 

```bash
git clone https://github.com/FrankSteps/pov-digital-circus
cd pov-digital-circus
make
make run 
```
